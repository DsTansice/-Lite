    你这个学期必须选修 numCourse 门课程，记为 0 到 numCourse - 1 。在选修某些课程之前需要一些先修课程。 例如，想要学习课程 0 ，你需要先完成课程 1 ，我们用一个匹配来表示他们：[0, 1]。给定课程总量以及它们的先决条件，请你判断是否可能完成所有课程的学习？

    示例 1:

    输入: 2, [[1, 0]]
    输出: true

    • 左括号必须用相同类型的右括号闭合。
    • 左括号必须以正确的顺序闭合。

    注意： 空字符串可被认为是有效字符串。

    示例 1:

    输入: “()”
    输出: true

    示例 2:

    输出: true

    示例 3:

    输入: “(]”
    输出: false

    示例 4:

    输入: “([)]”
    输出: false

    示例 5:

    输入: “{[]}”
    输出: true
